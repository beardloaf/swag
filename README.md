# swag

A visual contact sheet of **40 design-forward employee-swag and anniversary-gift ideas** — everything beyond the usual tee/tote, capped at $1,000, curated with an Uncrate / cool-hunting eye across four tiers (everyday · elevated · travel · milestone).

**Live page:** https://beardloaf.github.io/swag/

## How to use it
1. Open the live link.
2. Type your name in **“Add your name.”** You get an initial-in-a-circle in a color of your own.
3. **Click any tile** to tag the items you're excited about — your initials appear in the top-right corner. Click again to un-tag.
4. Hover a tile for the pitch; **“View →”** opens the store page.
5. Hit **“Copy share link”** and send it around — everyone shares **one live board**, so picks sync automatically (a small ● status shows live / offline).

## Notes
- Tags sync live to a shared JSON store (no logins). Everyone who opens the link sees the same board within a few seconds. Your name is remembered per-browser via `localStorage`.
- Prices are ballpark retail and exclude branding/bulk discounts. Most brands run corporate gifting programs; Reachdesk / Sendoso / SwagUp can warehouse & ship on demand.
- All product images are hot-linked from the brands' own CDNs.
